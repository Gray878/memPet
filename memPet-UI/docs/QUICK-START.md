# memPet UI 快速启动指南

## 第一次运行

### 1. 安装 Node.js

确保已安装 Node.js (推荐 v18 或更高版本):

```bash
node --version
```

### 2. 安装依赖

```bash
cd memPet-UI
npm install
```

这将安装所有必要的依赖包,包括:
- React 18
- Electron
- Tailwind CSS
- Zustand
- Lucide React

### 3. 启动开发服务器

```bash
npm run dev
```

这将:
1. 启动 Vite 开发服务器 (端口 5173)
2. 启动 Electron 应用
3. 自动打开应用窗口

## 常见问题

### Q: 启动后看不到窗口?
A: 检查终端是否有错误信息,确保所有依赖已正确安装。

### Q: 热重载不工作?
A: 修改渲染进程代码会自动热重载,修改主进程代码需要重启应用。

### Q: 如何查看开发者工具?
A: 按 `Ctrl+Shift+I` (Windows/Linux) 或 `Cmd+Option+I` (macOS)

### Q: 宠物动画是临时的?
A: 是的,当前使用简单的圆形占位符,后续会集成 Live2D。

### Q: 对话功能不工作?
A: 当前只是前端界面,没有连接真实的 LLM API,消息会显示但不会有回复。

### Q: 记忆浏览器是空的?
A: 当前没有真实数据,需要集成 memPet-server 后才会有记忆数据。

## 下一步开发

如果你想继续开发,建议按以下顺序:

1. **实现 Electron 主进程** (`src/main/index.ts`)
   - 配置窗口属性 (透明,无边框,置顶)
   - 实现系统托盘
   - 添加全局快捷键

2. **建立 IPC 通信** (`src/main/ipc/`)
   - 定义 IPC 通道
   - 实现消息处理器
   - 连接前端和后端

3. **集成 memPet-server** (`src/main/services/`)
   - 实现 API 客户端
   - 添加错误处理
   - 实现离线队列

4. **添加系统监控** (`src/main/utils/`)
   - 活动窗口检测
   - 工作时长统计
   - 状态分析

5. **集成 LLM API**
   - 实现 Claude/OpenAI 客户端
   - 添加流式响应
   - 实现上下文管理
