# memU-server 快速启动指南

## 第一步：配置 .env 文件

打开 `.env` 文件，填入你的配置：

```env
# 修改这里：填入你的通义千问 API Key
OPENAI_API_KEY=sk-你的API_Key

# 其他配置保持默认即可
OPENAI_BASE_URL=https://dashscope.aliyuncs.com/compatible-mode/v1
DEFAULT_LLM_MODEL=qwen-plus
DATABASE_URL=postgresql+psycopg://postgres:postgres@localhost:5432/memu
STORAGE_PATH=./data
```

## 第二步：启动数据库

```powershell
docker compose up -d postgres
```

等待 3-5 秒让数据库启动。

## 第三步：启动服务

```powershell
uv run fastapi dev
```

## 常见问题

### Q: 提示 "OPENAI_API_KEY 未设置"
A: 编辑 `.env` 文件，填入你的 API Key

### Q: 提示 "数据库配置不完整"
A: 先运行 `docker compose up -d postgres` 启动数据库

### Q: Docker 未安装
A: 下载安装 Docker Desktop: https://www.docker.com/products/docker-desktop
