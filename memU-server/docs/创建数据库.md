# 创建 PostgreSQL 数据库

在启动 memU-server 之前，需要先创建数据库。

## 方法 1: 使用 psql 命令行

```bash
# 连接到 PostgreSQL
psql -h 127.0.0.1 -U postgres

# 在 psql 中执行
CREATE DATABASE memu;

# 退出
\q
```

## 方法 2: 使用 Python 脚本

```bash
uv run python tests/create_database.py
```

## 方法 3: 使用 pgAdmin 或其他图形工具

1. 打开 pgAdmin
2. 连接到 PostgreSQL 服务器
3. 右键点击 "Databases"
4. 选择 "Create" -> "Database"
5. 输入数据库名称: `memu`
6. 点击 "Save"

## 验证数据库创建

```bash
# 连接到数据库
psql -h 127.0.0.1 -U postgres -d memu

# 查看数据库
\l

# 退出
\q
```

## 下一步

数据库创建完成后，运行初始化脚本创建表：

```bash
uv run python tests/init_database.py
```

或者直接启动服务（会自动创建表）：

```bash
uv run fastapi dev app/main.py
```
